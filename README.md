Tommy Paolma | Computer Engineer Portfolio
Welcome to the personal portfolio website of Tommy Paolma, a skilled professional in computer engineering, software development, cybersecurity, web development, and data analysis. This site showcases his expertise, certifications, and contact information with an interactive design featuring particle animations and a typing effect.
Overview
This project is a static website built using HTML, CSS, and JavaScript. It includes:

A responsive layout optimized for desktop and mobile devices.
An interactive particle background using particles.js.
A dynamic typing effect displaying various professional titles.
A gallery of certifications with clickable previews.
Social media links and contact information.


Prerequisites

A modern web browser (e.g., Chrome, Firefox, Edge).
Internet connection (to load the particles.js library from CDN).

Installation

Clone or Download the Repository:

Download the ZIP file or clone the repository to your local machine:git clone <repository-url>


Alternatively, download the files manually and extract them.


Organize Files:

Place all files in the folder structure as outlined above.
Ensure all image files are in the images/ folder with the correct filenames.


Open the Website:

Navigate to the index.html file in your web browser by double-clicking it or using a local server (e.g., live-server for development).
The site should load with particle animations and the typing effect.



Usage

Navigation: Use the header menu to navigate between sections (Home, About, Certifications, Contact).
Certifications: Click on "View Certificate" links to preview certification images in a new tab.
Contact: Use the "Email Me" button to initiate contact (note: email is encoded for protection).
Social Links: Click the social media links in the footer to visit Tommy's profiles.

Customization

Update Content: Modify index.html to update text, images, or links.
Style Adjustments: Edit css/styles.css to change colors, fonts, or layout.
Particle Settings: Adjust js/particles.json to modify the particle animation (e.g., number, color, speed).
Typing Effect: Update the professions array in js/script.js to add or remove titles.

Dependencies

particles.js (loaded via CDN).
Font Awesome (for social icons, loaded via CDN).
